获取带货商品列表

查询达人橱窗中的商品。

查询商品

get https://www.bluvarri.com/api/open/affiliate/choice_product

返回给定达人的带货列表

Query 参数

influencer

string

必填
查看你的品牌ID

达人品牌ID，是带货达人的唯一标识。

chn

string

必填
查看你的渠道ID

分销渠道ID，比如独立网站或者APP。

current

integer

必填

分页查询当前页数。

size

integer

必填

分页查询返回列表的数据条数。

sort

string

可选

数据顺序，可选值：plan_price -- 单品价格，revshare -- 佣金比例，plan_sale_volume -- 单品销量，sales -- 销售额，views -- 浏览量，add_time -- 添加时间

返回响应

响应数据结构：商品列表数据结构

async function fetchProduct(influencer,chn,current,size,sort){
    const url="https://www.bluvarri.com/api/open/affiliate/choice_product?current="+current+"&size="+size+ "&influencer="+influencer+"&chn="+chn+"&sort="+sort;
    return await fetch(url);
}

async function loadProduct(){
    const response = await fetchProduct('达人的品牌Id','渠道Id',1,20,'add_time');
    var data = await response.json();
    console.log(data);
}

商品列表数据结构

商品列表数据对象。

code

integer

状态码：200 -- 成功，40023 -- 达人或渠道未配置正确。

message

string

请求状态消息。

product

object

商品对象

records

array

商品数据数组.

brandAvatar

string

品牌头像URL。

brandName

string

品牌名称。

thumbnail

string

商品封面URL。

productName

string

商品名称。

productPrice

string

商品价格。

affiliateLink

string

带货链接。

total

integer

数据总条数.

size

integer

数据列表条数。

current

integer

分页查询当前页数。

pages

integer

分页查询总页数。
